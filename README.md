# DarkBoss1BD_Network_Scanner_Tools

```
bash
git clone https://github.com/darkboss1bd/DarkBoss1BD_Network_Scanner_Tools.git
cd DarkBoss1BD_Network_Scanner_Tools
python DarkBoss1BD_Network_Scanner_Tools.py
```


```
bash
This Network Scanner tool includes:
Features:
Ping Scan - Checks if a host is alive
Port Scan - Scans common ports for open services
WHOIS Lookup - Shows domain registration information
DNS Lookup - Resolves domain names to IP addresses
Full Scan - Performs all scans sequentially
```
